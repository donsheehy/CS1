Getting Started with Git Bash (Windows)

Git Bash gives you a Unix-style command line on Windows — the same kind of shell your classmates on Mac and Linux are using. Follow these steps to install it and run your first commands.

1. Download

  1. Go to https://git-scm.com/download/win.
  2. The download should start automatically. If it doesn't, click the link for the 64-bit Git for Windows Setup.

2. Install

  1. Open the file you downloaded (something like Git-2.x.x-64-bit.exe).
  2. Click through the installer. The default options are fine — you do not need to change anything. Just keep clicking Next, then Install.
  3. When it finishes, uncheck any "View release notes" boxes and click Finish.

You do not need administrator rights or a reboot.

3. Open Git Bash

  • Click the Start menu, type Git Bash, and press Enter.
  • A black terminal window opens with a prompt ending in $. This is your shell.

Tip: You can also right-click inside any folder in File Explorer and choose Open Git Bash here to start in that folder.

4. Try your first commands

Type each of these and press Enter:

pwd           # print the folder you're currently in
ls            # list files in the current folder
cd Desktop    # move into your Desktop folder
mkdir cs1     # make a new folder called cs1
cd cs1        # move into it

To go back up one folder, use cd ...

5. Check that everything works

Run:

git --version

You should see a version number (for example, git version 2.45.0). If you do, you're all set.

Quick reference

Command What it does
pwd Show the current folder
ls List files
cd X Move into folder X
cd .. Move up one folder
mkdir X Make a new folder X
clear Clear the screen

Running Python in Git Bash

Python is not included with Git Bash — install it separately.

  1. Download it from https://www.python.org/downloads/.
  2. Run the installer. On the first screen, check the box "Add Python to PATH." This step is easy to miss and things won't work without it.
  3. Click Install Now and finish.

Close and reopen Git Bash, then check it worked:

python --version

You should see a version number (for example, Python 3.12.0).

To run a program, put your code in a file and run it:

python myscript.py

This works exactly the same as on Mac and Linux.

One quirk: the interactive Python prompt (typing just python with no file) may appear to hang in Git Bash. If that happens, use one of these instead:

winpty python    # or:
py

Running .py files is not affected — only the interactive prompt. The simplest habit is to always write your code in a file and run it with python file.py.

Set Git Bash as your VS Code terminal

If you use VS Code, you can make its built-in terminal use Git Bash so it matches these instructions.

  1. Open the terminal: View → Terminal (or press Ctrl+`).
  2. Click the dropdown arrow (˅) next to the + at the top-right of the terminal panel.
  3. Choose Select Default Profile.
  4. Pick Git Bash from the list.
  5. Click the + to open a new terminal — it will now be Git Bash.

VS Code finds Git Bash automatically once Git for Windows is installed.

Getting help

If a command doesn't work, first check for typos — the shell is case-sensitive and spaces matter. If you're still stuck, bring the exact command you typed and the error message to class or office hours.
