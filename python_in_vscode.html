Running Python in VSCode

For a first-time setup, the simplest path is:

1. Install Python from python.org. On Windows, check "Add Python to PATH" during install. This is the one step people skip and then nothing works.

2. Install the Microsoft Python extension in VSCode (Extensions sidebar → search "Python" → install the one by Microsoft).

3. Open a folder, not just a loose file (File → Open Folder). VSCode behaves better when it has a folder to work in.

4. Run the code by opening a .py file and clicking the ▶ Run button in the top-right corner. That's it — output appears in the integrated terminal.

The Run button is the key thing for beginners. It handles selecting the interpreter and running the file so they don't have to touch the command line or learn python filename.py yet.

Two things worth flagging to avoid early frustration: if the Run button doesn't appear or errors, they may need to pick an interpreter (bottom bar, or Ctrl+Shift+P → "Python: Select Interpreter"). And students who install from the Microsoft Store version of Python sometimes hit permission oddities — the python.org installer is more predictable for a class.
