Write a function that turns "military time" into am/pm. The input should be a string "xx:yy" where xx is the two digit hour and yy is the two digit minute.

Click to reveal. 
def am_pm(t):
    hours, minutes = t.split(':')
    hours = (int(hours) - 1) % 12 + 1 

    return f"{hours}:{minutes}"

print(am_pm("0:40"))
