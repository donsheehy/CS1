Intro to Python — Topic Sequence

A teaching order where each unit depends only on earlier ones. String manipulation is placed after loops, so every string skill lands right when students have a use for it.

1. Getting started

  • Running Python (REPL, scripts, print())
  • Comments (#)
  • input() for user input

2. Variables and basic types

  • Variables and assignment (=)
  • Numbers: int, float; arithmetic (+ - * / // % **)
  • Strings basics only: quotes, concatenation, f-strings, len()
  • type() and type conversion (int(), float(), str())
  • Booleans (True / False)

3. Conditionals

  • Comparison operators (== != < > <= >=)
  • if / elif / else
  • Logical operators (and, or, not)

4. Loops

  • while loops
  • for loops and range()
  • break and continue
  • Accumulator patterns (counters, sums)

5. Strings in more depth

  • Indexing and slicing (s[0], s[1:4], negative indices)
  • Looping over characters
  • Methods: .lower(), .upper(), .strip(), .split(), .replace(), .find()

6. Lists

  • Creating, indexing, slicing (reuses string indexing — now familiar)
  • Methods: .append(), .pop(), .insert(), .remove(), .sort(), len(), in
  • Iterating over lists
  • Nested lists

7. Functions

  • def, parameters, return
  • Arguments vs. return values
  • Scope (local vs. global)
  • Default parameters

8. More data structures

  • Tuples
  • Dictionaries: keys/values, .get(), .keys(), .items(), iterating
  • Sets (brief)

9. Working with data

  • List comprehensions (basic)
  • Reading/writing files (open(), with, .read(), .readlines())

10. Errors and robustness

  • Reading tracebacks
  • try / except

11. Organizing code

  • Importing modules (math, random, datetime)
  • Writing/importing your own modules

12. Optional stretch topics

  • Classes and objects (class, __init__, methods)
  • enumerate(), zip()
  • Lambda / map / filter
